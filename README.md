## ¿Que es Dynamo Reflex?
<p>Dynamo Reflex, es un bot inteligente de Discord, desarrollado con DiscordJS y la API de CHAT GPT 3.5, el cuál, responde las consultas que tu indiques en el campo correspondiente.</p>
